# cli-with-node
cli using node

requirements:

- install NodeJS
- install NPM
- `sudo apt-get install node-gyp`
- `sudo npm install -g pkg`

Run:
```
# set ENV:
export PACKAGE_ENV=dev
npm start
```

install:
```
# locally
npm install --global
# other devices
npm install --global outside-cli
```

build:
```
pkg
```
